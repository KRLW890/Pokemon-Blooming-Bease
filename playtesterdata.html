Pokemon Blooming Beast automatically records information, such as the results of your battles, into the file "playtesterdata.txt". For transparency, an exhaustive list of the information recorded is provided here:
  • Starter Pokemon choice,
  • Each area entered,
  • Each wild Pokemon encountered, along with its level and result (this includes captured Pokemon),
  • Each trainer battled, along with the result,
  • The levels of your Pokemon at specific points,
  • Each overworld item obtained, along with items from NPCs,
  • Each time healed and where,
  • Each time interacting with a shop and where,
  • Each in-game trade,
  • Each time saved along with the version number and in-game play time,
  • And each time the game is loaded, along with the version number.
These allow the developer to reconstruct the playtester's experience in lieu of being able to watch.
Anything not expressly listed above is not recorded.

Submitting the playtester.txt file is not required in the playtester survey, but it is recommended.

Thank you for playing Pokemon Blooming Beast.
